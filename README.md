# DevJams'19 

Repository for the official website of DevJams 2019, flagship event of DSC VIT.

## To run locally

    $ git clone https://github.com/nikhils4/Devjams-2019.git
    $ cd Devjams-2019

Website is live [here](https://devjams.dscvit.com "DevJams Website")
